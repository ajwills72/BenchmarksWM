# sixproblems
R data package for benchmark experiments in working memory research. 
## Installation
From within R, use the following commands:
```
install.packages("devtools")
library(devtools)
install_github("ajwills72/BenchmarksWM")
library(BenchmarksWM)
```
## Help
For help, type:
```?sixproblems```
